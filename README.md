# Zihao Wang Personal Homepage

This repository hosts the personal academic homepage of Zihao (Matthew) Wang.

Live site:
https://zihaowang-kiaa.github.io/personal-homepage/

## Contents

- `index.html` - homepage
- `pages/high-redshift-black-holes-agns.html` - project page
- `pages/posts.html` - posts, collections, albums, and lecture notes
- `source materials/` - visual assets used by the pages
- `albums/` - photography albums shown on the posts page
- `Zihao_2023(Eng).pdf` - CV linked from the homepage

## Local Preview

Run a simple static server from the repository root:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/
```

## Deployment

The site is intended to be served with GitHub Pages from the `main` branch root.
